Simple .NET6 API using GraphQL.

# Technologies&Features
- .NET 6 webapi
- VSCode
- Docker
- MSSQL
- GraphQL
  - Query
  - Mutation
  - Subscription
  - Filtering
  - Sorting
- HotChocolate
- GraphQL Voyager
- Entity Framework (Code First)
- WebSockets
